# MCMR-Data Initiative

**Mission:** Improve maternal and child health through data-driven malaria response in Edo State, Nigeria.

**Goals:**
- Collect and analyze malaria-related health data
- Support health workers and decision-makers with actionable insights
- Build a community of health and data professionals

**How to Contribute:**
- Suggest improvements via Issues
- Share data analysis scripts
- Collaborate on health impact projects
